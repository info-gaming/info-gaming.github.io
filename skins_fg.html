Fall Guys - Les skins

Voici une liste des skins présents sur Fall Guys, il y a également leur prix et leur rareté. Ces skins sont répartis dans 3 catégories :

  • • Les skins pouvant être achetés dans le magasin

Si vous recherchez un skin en particulier, appuyez sur une de ces 3 catégories pour vous y rendre plus vite ! Si vous voulez regarder tous les skins, défilez simplement vers le bas.
